1.	Сосчитать количество клиентов, покупающих товар.

SELECT COUNT(Client.Client)
FROM Client

2.	Сколько посредников поставляют товар для фирмы.

SELECT COUNT(c.Client)
FROM Client c
WHERE c.Status LIKE 'посредник'

3.	Вывести среднюю сумму сделки по проданному товару.

SELECT AVG(cg.CostUnit*cg.Count) AS 'Средняя сумма за проданные товары'
FROM Client c, ClientGoods cg


4.	Посчитать на какую сумму было всего поставлено товара.

SELECT sg.CostUnit*sg.Count AS 'Сумма поставленного товара'
FROM SallerGoods sg


5.	Определите максимальное количество закупленного товара шт.(кг)

SELECT MAX(cg.Count)
FROM ClientGoods cg

6.	Какая минимальная сумма покупки, на которую клиент сделал заказ.

SELECT MIN(cg.CostUnit*cg.Count)
FROM ClientGoods cg

7.	Посчитать на какую сумму было продано товара магазинам.

SELECT SUM(cg.CostUnit*cg.Count)
FROM ClientGoods cg, Client c
WHERE c.INNClient=cg.INNClient AND c.Status LIKE 'посредник'


8.	Сосчитать общее количество видов закупленных фирмой вафель.

SELECT COUNT(sg.Goods)
FROM SallerGoods sg
WHERE sg.Goods GLOB 'Вафли*'

9.	Выведите среднюю цену на растительное масло (закупочную).

SELECT AVG(sg.CostUnit)
FROM SallerGoods sg
WHERE sg.Goods GLOB 'масло подсолнечное*';

10.	Определите сколько всего было продано килограмм яблок.

SELECT SUM(cg.Count)
FROM SallerGoods sg, ClientGoods cg
WHERE sg.ID=cg.ID AND sg.Goods GLOB 'Яблоки*';

11.	Определите какое количество картофеля закупили кафе.

SELECT SUM(cg.Count)
FROM Client c, ClientGoods cg, SallerGoods sg
WHERE c.INNClient=cg.INNClient AND cg.ID=sg.ID AND c.Status LIKE 'кафе' AND sg.Goods GLOB 'Картофель*'

12.	Посчитать сколько клиентов купили перец черный молотый.


SELECT COUNT(c.Client)
FROM Client c, ClientGoods cg, SallerGoods sg
WHERE c.INNClient=cg.INNClient AND cg.ID=sg.ID AND sg.Goods GLOB 'Перец черный молотый'


13.	Определить сколько наименований товара было куплено у посредников.

SELECT sg.Goods
FROM Seller s,SallerGoods sg
WHERE s.INNSeller=sg.INNSeller AND s.Status LIKE 'посредник'

14.	Определить минимальную цену за единицу проданного товара.

SELECT MIN(cs.CostUnit)
FROM ClientGoods cs


15.	Определите минимальную сумму на какую было закуплено товара.

SELECT MIN(cs.CostUnit*cs.Count)
FROM ClientGoods cs


16.	Определить максимальную сумму за товар, проданный магазинам.


SELECT MAX(cs.CostUnit*cs.Count)
FROM ClientGoods cs, Client c
WHERE cs.INNClient=c.INNClient AND c.Status LIKE 'магазин'

17.	Определить со сколькими фирмами сотрудничает данная фирма.

SELECT COUNT(s.Seller)
FROM Seller s

18.	Определить минимальную сумму сделки по закупке товара у производителей.


SELECT MIN(cs.CostUnit*cs.Count)
FROM ClientGoods cs, Client c, SallerGoods sg, Seller s
WHERE cs.INNClient=c.INNClient AND cs.ID=sg.ID AND sg.INNSeller=s.INNSeller AND s.Status LIKE 'производитель'


19.	Определить среднюю цену за пачку чая при покупке.

SELECT AVG(cs.CostUnit)
FROM ClientGoods cs, Client c, SallerGoods sg, Seller s
WHERE cs.INNClient=c.INNClient AND cs.ID=sg.ID AND sg.INNSeller=s.INNSeller AND sg.Goods GLOB 'Чай*'


20.	Определите максимальную сумму прибыли за товар, проданный посреднику.

SELECT MAX((cg.CostUnit*cg.Count)-(cg.Count*sg.CostUnit))
FROM ClientGoods cg, Client c, SallerGoods sg
WHERE sg.ID=cg.ID AND c.INNClient=cg.INNClient AND Status LIKE '%посредник%'

21.	 Определите минимальную прибыль за товар, купленный у посредника.

SELECT MIN((cg.CostUnit*cg.Count)-(sg.CostUnit*cg.Count))
FROM ClientGoods cg, Client c, SallerGoods sg
WHERE sg.ID=cg.ID AND c.INNClient=cg.INNClient AND Status LIKE '%посредник%'

25.	 Определите среднюю прибыль от перепродажи масла.

SELECT AVG((cg.CostUnit*cg.Count)-(sg.CostUnit*sg.Count))
FROM ClientGoods cg, Client c, SallerGoods sg, Seller s
WHERE cg.INNClient=c.INNClient AND cg.ID=sg.ID AND sg.INNSeller=s.INNSeller AND sg.Goods GLOB 'масло*' 


26.	 Со сколькими посредниками сотрудничает фирма.

SELECT DISTINCT s.Seller
FROM ClientGoods cg, Client c, SallerGoods sg, Seller s
WHERE cg.INNClient=c.INNClient AND cg.ID=sg.ID AND sg.INNSeller=s.INNSeller AND s.Status LIKE 'посредник'
